उबुन्तू ११.०४ मध्ये देवनागरी लेखन

Mon 03 October 2011

मी मागे टेक मराठीच्या मेळव्यामधे "संगणक वापरून मराठी typing कसे करावे" ह्या बद्दल एक प्रात्यक्षीक घेतले होते. (त्याच्या slides तुम्हाला इथे मिळतील) तेंव्हा मी फक्त Windows किंवा browser-based software बद्दलच बोललो होतो. त्या नंतर मी विशाल तेलन्ग्रे ह्यांच्या लेखावरून (link) Ubuntu 10.10 वर baraha वापरून देवनागरी लिहू शकलो. पण जेंव्हा मी Ubuntu 11.04 ला upgrade केले, तेंव्हा त्या सूचना उपयोगी पडल्या नाहीत (ubuntu मधल्या अंतर्गत बदलांमुळे हे घडले असावे असा अंदाज)मग मी PLUG च्या mailing list वर ह्या विषयावर चर्चा केली तेंव्हा माझा हा problem सुटला.

त्या चर्चेचा इतरांना फायदा व्हावा म्हणून ह्या सूचना इथे देत आहे . . . .

प्रथम आवश्यक ते software install करण्या साठी :

sudo apt-get install ibus-m17n ttf-devanagari-fonts
  1. नंतर, Alt-F2 करून, ibus-setup टाईप करावे.
  2. मग, विशाल तेलन्ग्रे ह्यांच्या लेखामधे दिल्या प्रमाणे, मराठी भाषा select करावी.
  3. मराठी मधला itrans हा पर्याय निवडायला हरकत नाही (मी आधी, baraha साठी प्रयत्न करत होतो, पण नंतर लक्षात आले की itrans हा सुद्धा उत्तम् पर्याय आहे  - ह्या सूचने बद्दल - अभिषेक आंबेरकर ह्यांना धन्यवाद)
